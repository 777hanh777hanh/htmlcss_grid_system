# 06. Xây dựng grid system row-cols

`row-cols` dùng để xác đinh mỗi hàng sẽ có bao nhiêu cột (col)

## Triển khai

`row-cols-*` phải đặt trước `.col-*` để **`.col-*`**` được ưu tiên hơn`.

```css
.row-cols-1 > * {
    flex: 0 0 auto;
    width: calc(100% / 1);
}
.row-cols-2 > * {
    flex: 0 0 auto;
    width: calc(100% / 2);
}
/* ... */
.row-cols-12 > * {
    flex: 0 0 auto;
    width: calc(100% / 12);
}

.col-1 {
    /* ... */
}
.col-2 {
    /* ... */
}

/* ... */
```
